----------------backend-----------------------------

comando para instalar Express para manejar el servidor y pg para conectar con posgrest
npm install express pg

Instalacion de nodamon como dependencia de desarrollo para el reinicio automatico
del servidor durante el desarollo
npm install nodemon --save-dev

y en las dependencias del archivo package.json ponemos una , y "start": "nodemon index.js"
esto para poder usar el nodemon

creacion de la base de datos

  id SERIAL PRIMARY KEY,
  nombre VARCHAR(100),
  correo VARCHAR(100),
  edad INT

para este paso debemos tener ya el node.js instalado y para instalar angular CLI ejecutamos el siguiente comando
npm install -g @angular/cli

comando para crear servicio para gestionar peticiones http

ng generate service services/user


ahora la creacion de un nuevo componente en angular para poder mostrar los usuarios

ng generate component components/user-list
